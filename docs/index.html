Isi Kandungan Modul Pembelajaran

Senarai Modul

  • Modul 1: Persediaan Laravel & Asas Kerangka
    Hasil Pembelajaran:

    Pelajar dapat menyediakan persekitaran pembangunan Laravel sepenuhnya serta memahami struktur asas MVC, routing, dan Blade templating.

    Kandungan Modul:
    • Pemasangan XAMPP, Composer, Node.js, Git, dan Visual Studio Code
    • Penciptaan projek Laravel menggunakan Composer
    • Konfigurasi fail .env dan pelaksanaan migrasi awal
    • Pengenalan kepada konsep Model-View-Controller (MVC) dalam Laravel
    • Penggunaan routing asas melalui fail web.php
    • Penggunaan Blade templating dan penyediaan layout asas
    • Pemasangan Laravel Bootstrap Auth Scaffolding dan Spatie Laravel Permission
    Aktiviti Pembelajaran:
    • Menyediakan projek Laravel
    • Ujian login dan navigasi asas sistem
    Penilaian:
    • Sistem Laravel dapat dijalankan dengan login berfungsi
    • Struktur folder dan kod difahami oleh pelajar
  • Modul 2: Antaramuka Dinamik dan CRUD Interaktif
    Hasil Pembelajaran:

    Pelajar dapat membina antaramuka pengguna yang dinamik menggunakan JavaScript serta melaksanakan operasi CRUD secara interaktif menggunakan AJAX.

    Kandungan Modul:
    • Pembinaan borang pelajar dengan pengesahan input menggunakan JavaScript
    • Interaksi masa nyata seperti pratonton data dan medan bersyarat
    • Struktur CRUD Laravel: model, migration, controller, view
    • Penggunaan modal Bootstrap untuk tambah dan kemaskini data
    • Penggunaan AJAX Fetch API untuk tambah, kemaskini dan hapus data tanpa 'page reload'
    Aktiviti Pembelajaran:
    • Membina senarai pelajar interaktif
    • Mengintegrasikan AJAX ke dalam operasi CRUD melalui modal
    Penilaian:
    • Semua operasi CRUD berfungsi tanpa muat semula halaman
    • Interaksi dinamik diuji secara manual dan berfungsi sepenuhnya
  • Modul 3: Pengujian Unit menggunakan PHPUnit / Pest
    Hasil Pembelajaran:

    Pelajar dapat menulis dan menjalankan ujian unit terhadap model dan controller dalam Laravel menggunakan PHPUnit atau Pest.

    Kandungan Modul:
    • Pengenalan kepada perintah php artisan test
    • Pemasangan dan konfigurasi asas Pest (jika digunakan)
    • Penulisan ujian unit untuk pengesahan model, logik controller, dan capaian route
    Aktiviti Pembelajaran:
    • Membina fail ujian bagi penyimpanan data pelajar
    • Menjalankan dan mengesahkan keputusan ujian
    Penilaian:
    • Minimum 3 ujian unit dijalankan dan lulus
    • Pelajar memahami asas pembangunan berasaskan ujian (TDD)
  • Modul 4: Penjana Dokumentasi Tersuai menggunakan OpenAI
    Hasil Pembelajaran:

    Pelajar dapat menjana dokumentasi sistem secara automatik menggunakan OpenAI dan memaparkannya dalam antaramuka web.

    Kandungan Modul:
    • Pembinaan Artisan Command: generate:docs
    • Imbasan fail controller dan view untuk mengenal pasti struktur dan logik
    • Integrasi OpenAI API untuk menjana dokumentasi dalam format JSON
    • Penyimpanan hasil dokumentasi ke dalam jadual pangkalan data (documentation_entries)
    • Paparan dokumentasi melalui templat Blade pada halaman /docs
    Aktiviti Pembelajaran:
    • Membangunkan perintah Laravel tersuai
    • Mempratonton dokumentasi dalam antaramuka templat sedia ada
    Penilaian:
    • Dokumentasi dalam format JSON berjaya disimpan ke dalam pangkalan data
    • Halaman /docs dapat dibaca dan digunakan dengan baik oleh pengguna akhir
